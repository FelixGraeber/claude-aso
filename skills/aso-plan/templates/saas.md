# SaaS / Productivity ASO Template

## Key Metrics
- Average conversion: 20-30%
- Average rating threshold: 4.3+
- Description importance: HIGH (users read before committing)

## Keyword Themes
- Function terms: project management, task tracker, notes, calendar, CRM
- Benefit terms: productivity, organize, collaborate, automate, sync
- Integration terms: cloud, offline, team, share, export
- Comparison terms: alternative to [competitor], best [function] app

## Screenshot Strategy
- Clean, professional UI showcase
- Feature workflows (before/after)
- Cross-device sync demonstrations
- Dashboard/analytics views
- Collaboration features

## Monetization-ASO Alignment
- Subscription apps: emphasize free trial in promotional text
- Highlight team/enterprise plans
- Use "free" strategically if freemium model

## Category Competition
High — target specific use-case long-tail keywords rather than broad terms.
