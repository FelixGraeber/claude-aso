# Gaming ASO Template

## Key Metrics
- Average conversion: 30-45%
- Average rating threshold: 4.2+
- Screenshot importance: VERY HIGH (visual-first category)

## Keyword Themes
- Genre terms: puzzle, strategy, RPG, action, casual, idle, clicker
- Mechanic terms: match-3, tower defense, battle royale, multiplayer
- Audience terms: offline, free, no wifi, kids, family
- Quality terms: HD, 3D, realistic, pixel

## Screenshot Strategy
- Show gameplay (not menus)
- Action/exciting moments
- Character showcase
- Progression/achievement systems
- Multiplayer/social features

## Monetization-ASO Alignment
- If free-to-play: emphasize "free" in metadata
- If premium: emphasize quality/value in screenshots
- IAP naming: use as additional keyword surface (iOS)

## Category Competition
Very High — differentiation through visuals and long-tail keywords is critical.
